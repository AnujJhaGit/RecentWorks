What can this tool do?

  • Use this tool to extract phone numbers from web pages and data files.
  • The output is one or more columns of phone numbers. You can see the output below or as an Excel file.

What are my options?

  • Optionally input a list of web pages to scan.
  • You can choose the number of phone numbers per line (default is 1).
  • You may choose the output delimiter for multi-column output or use a comma (the default).
  • You can remove duplicate phone numbers.
  • You can sort the phone numbers.
  • Ignore phone numbers under a certain number of digits (see Output Options).
  • You can add a heading.
